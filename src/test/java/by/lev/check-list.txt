I. Calculator check list:
    1. 2 + 3 = 5.0
    2. 2 + -3 = -1.0
    3. -3 + 3 = 0.0
    4. -3.1 - -3.1 = 0.0
    5. -3.1 - 3.1 = -6.2
    6. 2 * 0.1 = 0.2
    7. 2 * -0.1 = -0.2
    8. 2 / 0 = ArithmeticException
    9. 2 / 0.1 = 20
    10. 2 / -0.1 = -20

II. InputConverter check list:
    1. Input: '-2+3' expect: -2+3
    2. Input: '(-2)-(-3)' expect: -2--3
    3. Input: '2*3=' expect: 2*3
    4. Input: '2 / 3' expect: 2/3
    5. Input: '(-2) / (+3) =' expect: -2/+3

III. InputChecker check list:
    1. Input: '3.0+-2.', method: checkInput(String line), expect: true.
    2. Input: 'word', method: checkInput(String line), expect: false.
    3. Input: '3.0/0.1', method: checkNotByZero(String line), expect: true
    4. Input: '3.0/0.', method: checkNotByZero(String line), expect: false
    5. Input: '3.0/0', method: checkNotByZero(String line), expect: false
    6.  Method: checkLengthOfNumbers,
        args: String[] array = {"1234512345123451234512", "1", "1234512345123451234512"}

    9.  Input: Double.MAX_VALUE,

Результат не должен превышать значение Double.MAX_VALUE




Могут быть отрицательные числа
Могут быть числа в скобках
Выражение может быть записано с пробелами
Может быть без пробелов
В конце может быть равно, а может не быть

Не может (нельзя):
Делить на ноль
Огромные числа
Числа с двумя точками
Два знака +* и т.п.
Три операнда с двумя действиями
В принципе лишние число в строке
Первый операнд присутствует, действие, второго операнда нет

Разделить на 0., 0.23
