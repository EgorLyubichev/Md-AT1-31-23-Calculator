Выражения могут быть записаны в различных видах:

2+3

2 + 3 Допускается использование пробелов в выражении.

2.1+3.2 Допускается вычисление вещественных значений

2.-1. Числа могут быть с точкой но без остатка

-2--1 Числа могут быть отрицательными

(-2)*(-1) Числа могут быть в скобках

2/2= Допускается запись знака равно в конце примера

+2/+1 Допускается знак '+' перед значениями

На 0 делить нельзя

Максимальное значение вводимых чисел может быть 1_000_000_000_000.

Максимальное значение результата может быть не более 10_000_000_000_000.
